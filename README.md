# Details


This is a simple website made using nodejs.It will display top 5 songs of the year 2013 to 2017. 


## Dependencies Used



* Express
* ejs

## Installation 

* Clone This Repo And Open The Folder In Console
* Run `sudo npm install`
* Then `node app`
* Open `localhost:3000` in your Browser
* You Are Done!! :wink:
